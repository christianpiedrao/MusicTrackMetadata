# MusicTrackMetadata

Application that manage music track metadata

## Description

Sample project that manages music track metadata, it allows to retrieve data from Spotify API with the ISRC code and keep it on a local database. It also manages users and it is posible to register and later access. The project has two parts, the backend (Java + SpringBoot) and the frontend (React+Redux)

## Requirements

### Backend (Java)

- [Java](https://www.java.com/en/) >= JDK 17
- [Spring Boot](https://spring.io/projects/spring-boot) >= 4.x.x
- [Gradle](https://gradle.org/)
- Database: [MySQL](https://www.mysql.com/)

### Frontend (React)

- [Node.js](https://nodejs.org/en/) >= v10.0.0
- [npm](https://www.npmjs.com/)
- [React](https://reactjs.org/) >= 19.0.0
